Preview (masked result)

Steps:
1. Click "Choose Files" and select the equipped character sheets folder
2. Left-click drag to paint equipment area (green = keep)
3. Right-click drag to erase mistakes
4. Use "Copy Mask -> All Dirs" then fine-tune each direction
5. Green dot on direction tabs = has mask data
6. Click "Export All" to download overlay PNGs
7. Move PNGs to sprites/equipment/{slot}/{itemId}/male/
